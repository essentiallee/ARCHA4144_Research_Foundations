# Census data

This directory separates source files from reproducible outputs.

- `raw/nhgis/` contains authenticated IPUMS NHGIS downloads. These files are
  ignored by Git because IPUMS data should be redistributed only in accordance
  with its terms.
- `metadata/` contains a generated manifest describing the exact datasets,
  tables, variables, and boundary files used.
- `processed/` contains web-ready GeoJSON and a long CSV generated by Python.

No historical values in the website are hand-authored. Run the fetch and
prepare scripts described in the project README to produce them.
